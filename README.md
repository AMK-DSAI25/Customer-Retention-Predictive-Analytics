# Customer Retention Predictive Analytics

## Objective
To analyze customer data and build a machine learning model that predicts customer churn risk.

## Project Structure
- Data understanding and exploration
- Feature engineering and churn labeling
- Machine learning model development
- Model evaluation

## Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## Model
Logistic Regression was used as a baseline churn prediction model.

## Outcome
The project demonstrates a complete end-to-end data science workflow suitable for customer retention analysis.
